Essentials

The Dynamic CSS Reset Framework

Layout

Grids are the base structure of layout and the most essential tools for organizing information on the web. When done correctly the grid creates order, balance, and harmony to any website.

Container

The first piece for wrapping the page is the .container

<div class="container">
  <!-- Content here -->
</div>

Multiple .container can also be included on any single page.

<header class="container">
  <!-- Header content -->
</header>

<main class="container">
  <!-- Main content -->
</main>

<footer class="container">
  <!-- Footer content -->
</footer>

Grid System

The twelve-column grid offers the greatest flexibility for organizing information on the web, that it has become an essential tool for every web designer.

Wrapping

There are three options for wrapping the grid, the .large-grid, .medium-grid and the .small-grid. Each being a different tier for making responsive layouts.

Large Grid

<div class="large-grid">
  <!-- Columns here -->
</div>

Medium Grid

<div class="medium-grid">
  <!-- Columns here -->
</div>

Small Grid

<div class="small-grid">
  <!-- Columns here -->
</div>

Columns

Because the tier of the grid is declared with the wrapper all you need to do is to specify the number of columns the grid will .span-# and the finishing up with the .columns class.

<div class="large-grid">
  <article class="span-8 columns">
	<!-- Content here -->
  </article>

  <aside class="span-4 columns">
	<!-- Content here -->
  </aside>
</div>

Nesting

You cannot nest grids indefinitely that would be absurd, instead try to follow the sequence.

<div class="large-grid">
  <article class="span-12 columns">
	<div class="medium-grid">
	  <article class="span-6 columns">
		<div class="small-grid">
		  <article class="span-3 columns">
			<!-- Content here -->
		  </article>

		  <article class="span-3 columns">
			<!-- Content here -->
		  </article>
		</div>
	  </article>

	  <article class="span-6 columns">
		<div class="small-grid">
		  <article class="span-3 columns">
			<!-- Content here -->
		  </article>

		  <article class="span-3 columns">
			<!-- Content here -->
		  </article>
		</div>
	  </article>
	</div>
  </article>
</div>
